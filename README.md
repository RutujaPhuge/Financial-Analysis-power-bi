# Financial-Analysis-power-bi

Financial Analysis Dashboard- Power BI Project

This project showcases a comprehensive Financial Analysis Dashboard built using Power BI.The report provides actionable insights into sales trends, customer segments, product performance. Key features include:
  
   1: Sales Analysis by Month:-
       Visualizes monthly sales volume to identify high performing periods.
   
   2: Sales by Segment:-
       A pie chart breaking down sales across Goverment, Enterprise and Small Business sectors.

   3: Day Wise Sales, COGS and Profit Trend:-
       A Clustered column chart showing daily performance metrics.

    4: Top KPIs (Cards)-
         1) Top Selling product(Paseo)
         2) Total Number of Sales
         3) Gross Revenue(8.6 M)
         4) COGS Count


    5: Sales by Discount Band:-
        Donut chart illustrating the impact of discounts on total sales.

    6: Sales by Product:-
        A Funnel chart comparing product-wise sales.

    7: Sales & COGS by Country:-
        Area chart representing geographical sales and cost distribution.

    8: Interactive Country Slicer:-
        Enables dynamic filtering of the entire Dashboard by selected countries.

      Tools Used:-
        Power BI, DAX, Data Cleaning, Data Modelling, Visual Designs.

      Goal:-
         To Analyze Financial performance and derive Insights for Stratergic Business decisions.
